**Plump - Skincare Landing Page**

Uma landing page moderna e responsiva para divulgação de um sérum facial premium, desenvolvida com React, Vite, Tailwind CSS e React Router DOM.

Sobre o Projeto

Plump é uma aplicação web desenvolvida com foco em experiência do usuário, design moderno e navegação fluida. O projeto apresenta um produto de skincare de forma elegante, destacando seus ingredientes, benefícios e diferenciais através de uma interface intuitiva e responsiva.

Além da apresentação visual do produto, a aplicação conta com componentes interativos e formulários dinâmicos para proporcionar uma experiência mais envolvente ao usuário.

----------------------------------------------------------------------------------------------------------------------------------------------
 Funcionalidades

- Navegação SPA (Single Page Application) com React Router DOM
- Interface moderna e responsiva
- Formulário interativo com validações e feedback visual
- Apresentação dos ingredientes ativos
- Seção de benefícios do produto
- Navegação intuitiva entre páginas
- Componentes reutilizáveis
- Layout adaptado para dispositivos móveis e desktops
- Animações e transições suaves
